# Telegram Bot (Telethon)

## Overview
Ye ek simple Telegram bot hai jo Telethon library use karta hai.

## Features
- `/start` - Bot ko start karo
- `/help` - Help message dekho
- `/hello` - Bot hello bolega
- `/time` - Current date aur time dekho
- Echo feature - Private messages ko repeat karta hai

## Setup
Bot chalane ke liye aapko ye credentials chahiye:
1. **API ID** - Telegram se milta hai (my.telegram.org)
2. **API Hash** - Telegram se milta hai (my.telegram.org)
3. **Bot Token** - BotFather se milta hai (@BotFather on Telegram)
